# Sales Performance Dashboard (Power BI)

## Overview
This project analyzes sales performance, profitability, cost, and loss rate across multiple product categories and time periods. The dashboard was designed to support business decision-making through clear and interactive visualizations.

## Tools Used
- Power BI
- Microsoft Excel

## Key Metrics
- Total Sales
- Total Profit
- Total Cost
- Loss Rate (%)

## Key Insights
- Q1 recorded the highest quarterly sales compared to other quarters.
- Certain product categories contributed significantly more to profit.
- Monthly sales trends revealed seasonal fluctuations.
- Overall sales peaked in 2021 before declining in later years.
